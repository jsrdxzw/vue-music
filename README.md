# vue-music

based on vue-cli3.x and vue2.5x and vuex vue-router
___

## Project setup
```
npm install
```

### Start server
```
cd server
node index.js
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
